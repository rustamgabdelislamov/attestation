## Краткий обзор

1. Разработана онлайн платформу-торговой сети электроники
2. Реализована иерархия поставщиков
3. Пользователь с is_active=False не сможет работать через API, админ сможет отключить сотрудника при необходимости
4. Сотрудник не сможет убрать задолженность через API у компании перед поставщиком

### Запуск сервера

1. Установите Poetry 
pip install poetry
2. Войдите в виртуальное окружение
poetry shell
3. Установите зависимости
poetry install
4. Клонируйте репозиторий git@github.com:rustamgabdelislamov/attestation.git
5. Выполните команды
python manage.py csu - устанавливает суперюзера
python manage.py add_user - устанавливает пользователей для минимальной проверки
python manage.py runserver - запускает сервис
6. Для Postmana есть коллекция в которой все запросы подписаны и есть тело запросов
